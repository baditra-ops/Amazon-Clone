# Amazon Clone

A beginner-friendly Amazon homepage clone built using **HTML, CSS, and a little JavaScript**. This project was created during my **first semester at IIT (BHU) Varanasi** as part of my web development learning journey.

## About the Project

The goal of this project was to recreate the look and feel of Amazon's homepage while learning the fundamentals of front-end development. Through this project, I practiced building responsive layouts, designing navigation bars, creating product sections, implementing image sliders, and styling complex UI components using pure CSS.

## Features

* Amazon-inspired navigation bar
* Hero banner section
* Product cards and deal sections
* Image slider/carousel
* Multiple shopping categories
* Footer section similar to Amazon
* Hover effects and interactive UI elements
* Responsive layout using Flexbox

## Technologies Used

* HTML5
* CSS3
* JavaScript (Basic)

## What I Learned

* Structuring web pages with HTML
* Styling layouts with CSS
* Using Flexbox for positioning elements
* Working with images and background properties
* Creating sliders using JavaScript
* Building real-world UI clones
* Improving attention to design details

## Future Improvements

* Add full responsiveness for all screen sizes
* Implement a working search bar
* Add user authentication pages
* Create product pages
* Connect the project to a backend and database
* Improve animations and transitions
